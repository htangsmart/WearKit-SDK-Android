interface WKScanner

Helper for scan ble devices

For the scan process, it provides some additional implementations:

1.Filter devices.

2.Use different scanning modes for different scenes in the foreground and background.

For the foreground, use ScanSettings.SCAN_MODE_LOW_LATENCY. For the background, use ScanSettings.SCAN_MODE_LOW_POWER.

3.Optimize the exception of BleScanException.UNDOCUMENTED_SCAN_THROTTLE.
